1. '/' : 用來傳入任務判斷的值
格式:
modal : 
{
    type: int,
    goal: int
}

    type    |   項目    |   goal    |
     1      |   起床    |    --     |
     2      |   睡覺    |    --     |
     3      |   截圖    |    --     |
     4      |   散步    |    --     |   // 以上隨便丟一個值就好
     5      |   路程    |  distance |   // 增加的路徑
     6      |   發文    |     1     |
     7      |   留言    |     1     |
     8      |   等級    |     1     |   // 發文數、留言、等級提升1(除非你想升很多等之類的，不然就設1)

2. '/getdaily' : 用來獲取日常任務的完成情形
格式:
{
    wake: bool,
    sleep: bool,
    picture: bool,
    stroll: bool
}

3. '/getEXP' : 用來獲取玩家現在的經驗值
格式 : int
錯誤時回傳-1

4. '/getWalk' : 用來獲得和走路相關的資訊
格式 : 
{
    progress: int,    // 正在破哪個走路的徽章任務(全部徽章都拿到後會保持在最後一個徽章的要求)
    goal: int         // 累積路程
}
錯誤時所有值回傳-1
